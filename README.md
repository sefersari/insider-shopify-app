# insider-shopify-app

### Installation:
```
   composer install
   php artisan migrate
   
   php artisan serve
```

```
   Ready !!
```

#### Other:
```
    To log in, please register first,
    and while registering, please enter the company name.
    
    After that, enter the Shopify environment details from the settings menu.
    
    When you click the 'Sync Shopify Order' button under the Order menu,
    you will be able to list your Shopify orders.
    
    You can then browse through the pages to view the orders.
    
    Note: Currently, it is only available for the Shopify marketplace.
```
